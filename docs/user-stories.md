# User Stories — MVP

Chỉ các story cần cho MVP.

---

## Job applications

**US-1**  
As a job seeker, I want to add a new job application with company, position, JD link, job type, location, source, and optional salary/expected salary, so that I can track every role I apply to in one place.

**US-2**  
As a job seeker, I want to view a list of all my job applications, so that I can see my full pipeline at a glance.

**US-3**  
As a job seeker, I want to update an existing job application, so that I can fix or enrich details as I learn more about the role.

**US-4**  
As a job seeker, I want to update the pipeline status of an application (Applied, Screening, Interview, Offer, Rejected, On Hold), so that I know where each application stands.

**US-5**  
As a job seeker, I want to filter and sort applications by status, company, position, source, or apply date, so that I can quickly find the applications I care about.

---

## Notes

**US-6**  
As a job seeker, I want to add notes to an application (apply date, interview schedule, difficult questions, recruiter/hiring manager feedback), so that I can remember context for follow-ups and future interviews.

---

## Dashboard

**US-7**  
As a job seeker, I want to see a simple overview with total applications, counts per status, and total interviews conducted, so that I can gauge my job search progress without digging through each record.

---

## Deployment & access

**US-8**  
As a job seeker, I want the app deployed at a single public URL with a working API and responsive UI, so that I can use it daily and share the link on my CV for recruiters to view.
